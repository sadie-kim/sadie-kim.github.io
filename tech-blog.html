Explain control flow and loops using an example process from everyday life, for example 'waking up' or 'brushing your teeth'. (But not those ones).

Control flow and loops can be explained using our everyday routine. For example, wake up in the morning and go to work. Repeat until I no longer have the job. Instructions like this can be given to computer using the control flow and loops.

Describe what the DOM is and an example of how you might interact with it.

DOM is a web interface that allows us to access documents and elements so we can use them. DOM can be used to make functions work with JavaScript or add styling using CSS.

Explain the difference between accessing data from arrays and objects.

An array and an object are both used to store more than one values in a variable. However, array’s values can be accessed by referring to an index number. Where object’s values can be accessed by providing properties to access the stored value in an object.

Explain what functions are and why they are useful.

Functions are used in JavaScript to perform a particular task. When programming, often same tasks need to be repeated. When a function to perform the repeated task is defined, it simplifies the code.
